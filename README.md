This repository includes the source code of "**Adaptive Geometric Representation Learning through Mixed-Curvature Mixture-of-Experts Diffusion for Link Prediction**"

# Requirements

* Pytorch 
* python 3.x
* networkx
* scikit-learn
* scipy
* dgl
* torch_geometric

# How to run
* run `python -m main.run.train`
